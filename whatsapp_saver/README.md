# WhatsApp Saver

Flutter app to browse WhatsApp / WhatsApp Business status files and preview media.

## Features

- scans WhatsApp and WhatsApp Business `.Statuses` folders
- image and video tabs
- media preview experience
- Android permission handling for storage/media access

## Notes

- primarily intended for Android (depends on Android status folder locations)
- requires storage/media permissions on device

## Run

```bash
flutter pub get
flutter run
```

## Validate

```bash
flutter analyze
flutter test
```
